Input area commands

Enter / Ctrl+Enter Post message
Ctrl+Up/Down Go through local history
Up Compose message to the last private recipient
Esc Clear input area
/dc Disconnect from server
/rc Reconnect to server
/s %bot Subscribe to channel "bot"
/d %bot Unsubscribe from channel "bot"
